Техническое задание для интернет-магазина

Создание онлайн-платформы для продажи товаров

Описание проекта

Данный проект предполагает создание интернет-магазина, который будет предлагать широкий ассортимент товаров.

Основная цель — обеспечить удобный интерфейс для пользователей и эффективное управление товарами, а также прокачать скилы разработки на Django

Функциональные требования

  • 1. Регистрация и авторизация пользователей
      1. Запретить вход авторизированному пользователю
      2. Регаистрация/Вход (одно из двух )
      3. Имя/логаут (одно из двух)
      4. ополнить поля при регистрации(номер телефона как минимум)
  • 2. Личный кабинет пользователя
      1. Отдельная страница личного кабинета относящаяся ко всему проекту( не только магазин )
      2. В личном кабинете возможность редактировать существующие поля и сохранять
      3. Место для дополнений
  • 3. Корзина и оформление заказа
      1. Создать нормальный фронт дроп-даун корзины в навбаре, используя TABLE
      2. Кнопка "Перейти к оформлению" в навбар-корзине
      3. Помимо "заказать" добавить кнопку "Сохранить изменения" в корзине.хтмл
      4. Валидация полей: Оставить только натуральные значения
      5. Исправить ошибки JS в пересчете float стоимости
      6. Добавить больше полей-инпут для формления заказа и сделать автомаичесое заполение полей зарегестрированным пользователям
      7. После отправки формы заказа поблагодарить за заказ - при наличии в нем товаров
      8. "В корзине пока пусто" Если корзина пуста
      9. Место для дополнений
  • 4. Product-Deteil-View
      1. Добавить "похожие товары"
      2. Валидация полей
      3. Слайдер фотографий JS
      4. Добавить учет скидки в стоимость
      5. Место для дополнений
  • 5. Shop-home
      1. Добавить кнопку "купить сразу"
      2. Добавить учет скидки в стоимость
      3. Всплывающий баннер "заказать звонок"
      4. Добавить количество рзделам в зависимости от количества категорий товаров
      5. Добавить поиск по сайту
      6. Место для дополнений
  • 6. Адаптивность
      1. Добавить адаптивность к ГЛАВНОЙ СТРАНИЦЕ
      2. Добавить адаптивность к Странице резюме
      3. Место для дополнений
  • 7. Футеры
      1. Сделать человеческие футеры
      2. Место для дополнений
